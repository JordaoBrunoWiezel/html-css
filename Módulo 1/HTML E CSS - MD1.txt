Javascript
- interações -> menus /animações / popups/ validações



Estilo em HTML
<abertura de tag> conteúdo <fechamento de tag>
<h1> Exemplo de titulo </h1>
<p> Exemplo de parágrafo </p>
<img src="foto.png" alt="exemplo de foto">
<abertura de tag/ conteúdo / paramento /valor >


Estilo em CSS
h1 (seletor) { 
	(declaração)
	font-family: Arial;
	font-size: 20pt;
	color (propriedade) : blue; (valor)
}


Estrututa básica de documentação em HTML

<!DOCTYPE html> (vou desenvolver um site em html5)
<html lang="pt-br"> (vai possuir um conteudo em portugues)
	<head>
		<meta charset="UTF-8"> (meu site vai ser compativel com caracteres de UTF-8) 
		<meta charset="viewport"
		content="width=device-width, initial-scale=1.0"> (visulização em escala de 100%
		<title>Document</title>
	</head>
	<body>
		<h1>Olá, Mundo!</h1>
	</body>
</html>
